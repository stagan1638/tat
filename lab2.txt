Тестируемый продукт: интернет-магазин https://rockbastion.by
Тестовое окружение: десктопные и мобильные браузеры
---------------------------------------------------
        Тест-кейсы:
1.	Регистрация.
При вводе пустой строки пользователь должен быть проинформирован об этом.
	1) Пользователь нажимает на кнопку "Личный кабинет"
	2) Пользователь выбирает пункт "Регистрация" и нажимает на него.
	3) У пользователя должна открыться форма для ввода данных для регистрации.
  4) Пользователь заполняет поле "Пароль", "Подтвердите пароль" оставив поле "Email" пустым.
  5) Пользователь нажимает на кнопку "Продолжить"
  6) Пользователь должен увидеть надпись "Некорректный адрес электронной почты!"
   
Ожидаемый/реальный(ручная проверка на сайте) результаты совпали.
*******************
2.	Корректность работы добавления в "Корзина".
Пользователь должен иметь возможность добавлять товар в "Корзина"
	1) Пользователь в разделе "Одежда" нажимает на кнопку "В корзину" на товаре "Свитшот "2021 Classic Game""
	2) Товар должен оказаться в категории "Корзина".

Ожидаемый/реальный(ручная проверка на сайте) результаты совпали.
*******************
3.	Валидация строки поиска.
При вводе пустой строки пользователь должен быть проинформирован об этом.
	1) Пользователь нажимает на строку поиска.
	2) Пользователь нажимает клавишу "Enter" / кнопку с иконкой лупы.
	3) У пользователя должна отобразиться в строке надпись "Пожалуйста, введите поисковый запрос"

Ожидаемый/реальный(ручная проверка на сайте) результаты НЕ совпали:
	После шага 2) пользователя шаг 3) не выполняется.
*******************
4. Валидация написания e-mail в подписке на новости
В e-mail недопустимо использовать символы отличные от латинский букв, цифр, "." и "@".
	1) Неавторизованный пользователь на главной странице около надписи "Подписка" в строке "Введите Email, чтобы получать новости" вводит "1žyvie1blr@1.11". 
	2) Пользователь должен быть переадресован на страницу rockbastion.by/blog-news/subscribe
	3) Пользователь должен увидить надпись "Указан неверный адрес подписки."
  
Ожидаемый/реальный(ручная проверка на сайте) результаты НЕ совпали:
после шага 1) пользователь не был переадресован на страницу rockbastion.by/blog-news/subscribe

  3) После смены валюты все цены должны быть указаны в долларах.
